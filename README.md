# FTC DECODE 2025-2026 | InfotronX #19119

Competition robot code developed for the **FIRST Tech Challenge DECODE 2025-2026** season by **InfotronX #19119**.

This repository contains our robot software built in **Java** using **Android Studio** and the **FTC SDK**.  
It includes autonomous routines, TeleOp control, subsystems, testing, calibration, and trajectory-related development.

## Overview

This project was used during the FTC DECODE 2025-2026 season and reflects real competition work done under time pressure.

The codebase includes:
- **Autonomous routines** for different match scenarios
- **TeleOp control** for driver operation
- **Subsystem logic** for robot mechanisms
- **Sensor and camera integration**
- **Testing and calibration** OpModes
- **Trajectory and movement-related** files

Our work during this season contributed to earning the **“Nație Prin Educație” Award**.
